Description


Simple, yet comprehensive productivity app, featuring MultiRun, Checklist, Color Picker, Converter, Ruler, and many more tools you will (probably) need...


Acquisition




Features


  • MultiRun -- Select a number of apps, games, images, files, folders, etc., to open at once with a press of a button. Born to handle repetitive workflows!
  • Checklist
  • Color Picker
  • Converter
    • Length & Distance
    • Weight & Mass
    • Volume
    • Temperature
    • Data
    • Digital
    • Area
    • Energy
    • Pressure
  • Ruler
  • Auto Clicker
  • Decision Maker
    • Coin Flip
    • Lucky Wheel
    • Dice Roll
  • Tally
  • Options

Additional Notes


  • No internet connection is required.
  • The app is currently only supported on the Windows OS.
  • All save data is stored in the "saves" folder.
  • Version: 1.0-B
